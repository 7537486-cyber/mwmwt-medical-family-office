# Japan Medical Family Office

面向中国高净值家庭的日本 Medical Family Office 与 Longevity Concierge 官网。

## 本地预览

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`。

## 正式构建

```bash
npm run build
```

## 线上域名

正式域名：`https://mwmwt.com`

Vercel 绑定域名后，新网 DNS 建议添加：

```text
A      @      76.76.21.21
CNAME  www    cname.vercel-dns.com
```
